## **COURSE OVERVIEW**

This course is designed to introduce mechanical engineering students to the foundational concepts, tools, and techniques of machine learning. With hands-on coding assignments paired with theory sessions, practical projects, and an emphasis on mathematical rigor, this course offers a comprehensive learning experience tailored for engineers aspiring to excel in the rapidly evolving field of machine learning.

## **PROGRAM STRUCTURE** (24 Hours)

### **FOUNDATIONS (Sessions 1–11, 11 hours)**

**Session 1: AI Overview & Career** (1h)
- AI history, AI vs Data Science, need for ML in mechanical engineering
- ML categories (supervised, unsupervised, reinforcement learning)
- Career paths and opportunities in AI/ML for engineers

**Sessions 2–3: Python Fundamentals** (2h)
- Python syntax, data structures (lists, dicts, sets, tuples)
- Control flow (loops, conditionals), functions, NumPy basics

**Sessions 4–5: Mathematical Foundations** (2h)
- Linear algebra: vectors, matrices, dot products, norms
- Data concepts: NOIR classification, data structures

**Sessions 6–7: Statistics & Data Manipulation** (2h)
- Descriptive statistics: mean, variance, correlation
- Pandas DataFrames, groupby, merging, handling missing data

**Sessions 8–9: Data Analysis & Preparation** (2h)
- Exploratory Data Analysis (EDA) workflow
- Feature engineering: scaling, encoding, feature creation
- Churn dataset demo → complete data pipeline

**Sessions 10–11: Machine Learning Fundamentals** (2h)
- Supervised vs unsupervised learning, train/test/validation splits
- Evaluation metrics: accuracy, precision, recall, F1, cross-validation
- Scikit-learn API and workflow

### **ALGORITHMS (Sessions 12–20, 9 hours)**

**Sessions 12–13: Regression Techniques** (2h)
- Linear regression: least squares, gradient descent, regularization
- Logistic regression: sigmoid, probability interpretation
- Demos: housing prices, binary classification (Alice dataset)

**Sessions 14–15: Tree-based Methods** (2h)
- Decision trees: splitting criteria (Gini, entropy), overfitting, pruning
- Ensemble methods: bagging, boosting, random forests
- Demos: UCI Adults classification, heart disease prediction

**Session 16: Support Vector Machines** (1h)
- SVM theory: hyperplanes, margins, kernel trick
- When to use SVM, computational complexity
- Demo: MNIST digit classification

**Session 17: Naive Bayes** (1h)
- Bayes' theorem, conditional independence assumption
- Gaussian, multinomial, Bernoulli variants
- Application: text classification, sentiment analysis

**Session 18: K-Nearest Neighbors** (1h)
- Distance metrics (Euclidean, Manhattan), choosing k
- Lazy learning, computational considerations
- Demo: wine quality classification

**Sessions 19–20: Unsupervised Learning** (2h)
- K-means clustering: centroid initialization, elbow method, silhouette score
- Principal Component Analysis (PCA): variance explained, feature reduction
- Demos: customer segmentation, dimensionality visualization

### **PRODUCTION & INTEGRATION (Sessions 21–24, 4 hours)**

**Sessions 21–22: Model Evaluation & Hyperparameter Tuning** (2h)
- Classification metrics: confusion matrix, ROC-AUC, precision-recall
- Cross-validation strategies: k-fold, stratified splits
- Hyperparameter tuning: grid search, random search, model comparison

**Sessions 23–24: Capstone & Production** (2h)
- End-to-end ML pipeline: Titanic case study
- Model deployment: serialization, REST APIs
- MLOps essentials: versioning, monitoring, data drift, retraining
- Capstone assignment: design a production ML system
