# GNN Tutorial: ODSC West 2021 (Sujit Pal)

Python scripts derived from the ODSC West 2021 tutorial "Deep Learning with Graphs — an Introduction to Graph Neural Networks." Each script is self-contained, GPU-enabled, and downloads its own data.

## Environment Setup

Uses the `genai` conda environment with **CUDA 11.8** and **PyTorch 2.7**:

```bash
conda activate genai
```

To install from scratch:
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
pip install torch-scatter torch-sparse torch-cluster torch-geometric \
  -f https://data.pyg.org/whl/torch-2.7.0+cu118.html
pip install ogb scikit-learn matplotlib networkx
```

## Scripts

All datasets are auto-downloaded to `data/` on first run. Plots are saved to `plots/`.

| Script | Task | Dataset | Key Models |
|--------|------|---------|------------|
| `gnn_node_classification_cora.py` | Node classification | Cora (Planetoid) | GCN, GAT, GraphSAGE |
| `gnn_graph_classification_molhiv.py` | Graph classification | ogbg-molhiv (OGB) | GCN + mean/max/sum pooling |
| `gnn_link_prediction_cora.py` | Link prediction | Cora (RandomLinkSplit) | GCN + dot-product |
| `gnn_custom_message_passing.py` | Custom layer demo | Random synthetic | Custom multi-head GAT |
| `gnn_custom_dataset.py` | Custom dataset demo | Synthetic CSVs | RandomGraphDataset |

## Running

```bash
conda activate genai
cd Code/gnn/odsc2021-sujitpal

python gnn_custom_message_passing.py    # fast, no data download
python gnn_custom_dataset.py            # fast, generates synthetic data
python gnn_node_classification_cora.py  # ~5 min (500 epochs x 3 models)
python gnn_link_prediction_cora.py      # ~1 min (100 epochs)
python gnn_graph_classification_molhiv.py  # ~20 min (30 epochs, molhiv is large)
```

## Key Concepts

- **GCNConv** — Graph Convolutional Network (Kipf & Welling 2017)
- **GATConv** — Graph Attention Network (Velickovic et al. 2018)
- **SAGEConv** — GraphSAGE (Hamilton et al. 2017)
- **RandomLinkSplit** — Edge-level train/val/test split for link prediction
- **AtomEncoder** — OGB categorical atom feature → continuous embedding
- **MessagePassing** — Base class for custom convolution kernels
- **global_mean/max/add_pool** — Graph-level readout aggregation
