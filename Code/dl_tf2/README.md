# Deep Learning with TensorFlow 2

TF2/Keras tutorials from multiple contributors covering GANs, autoencoders, CNNs, RNNs, and Swift for TensorFlow examples.

## Setup
```bash
conda env create -f environment.yml
conda activate dl-tf2
```

## Contents

### GAN Variants (Udemy series)
- Fashion-MNIST DCGAN, Conditional GAN, Semi-Supervised GAN
- Each has both `.ipynb` and a matching `.py` script

### Swift for TensorFlow — Ayush series (`dl_s4tf_Ayush_*`)
8-part tutorial series (Tutorials 1–8) plus Autoencoder and GAN notebooks.
Note: Swift for TensorFlow is discontinued — these are for reference only.

### TF2 Fundamentals — Ashish series (`dl_tf2_ashish_*`)
Covers HelloWorld, NumPy integration, data pipelines, CNNs, text, training loops, RNNs, and Keras functional API.

### Other
- `dl_s4tf_google_basic_nn.ipynb` — Google's official basic NN example
- `dl_nn_udacity_StudentAdmissions.ipynb` — neural network from scratch (Udacity)
- `dl_nn-from-scratch.ipynb` — pure NumPy backpropagation implementation
- `dl_keras_2.1-a-first-look-at-a-neural-network_fchollet.ipynb` — Chollet chapter 2
