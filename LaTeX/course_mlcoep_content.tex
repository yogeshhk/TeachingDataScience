\begin{itemize}

\item Alice dataset (logistic regression)
\item UCI Adults (tree-based classification)
\item Heart disease (random forest)
\item MNIST digits (SVM classification)
\item Wine quality (K-NN)
\item Pima Indians diabetes (model evaluation)
\item Titanic (capstone case study)
\item Custom customer purchase data (K-means clustering)
\end{itemize}
